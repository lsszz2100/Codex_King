## 이 책은 누구를 위한 책인가

- "코드는 잘 모르지만 AI로 진짜 앱을 만들어 보고 싶다" — 초급자
- "Codex를 쓰긴 하는데 설정·MCP·스킬을 제대로 활용하고 싶다" — 중급자
- "에이전트 내부 동작을 이해하고 SDK로 직접 확장하고 싶다" — 고급자

세 부류 모두를 한 권으로 이어 줍니다. 터미널을 처음 여는 사람이 1번 절에서 출발해, 마지막에는 Codex의 Rust 코어 동작 원리를 이해하고 SDK로 자신만의 에이전트를 만드는 지점까지 도달하도록 설계했습니다.

## 이 책의 특징

1. GPT-5.5 최신 기준 — 2026년 6월 현재 가장 강력한 모델을 기준으로 작성했습니다.
2. 초급부터 천천히 — 터미널이 처음인 사람도 따라올 수 있게 단계별로 구성했습니다.
3. 실습 중심 — 거의 모든 장에 손으로 따라 하는 예제가 있습니다. `[실습]` 표시가 붙은 절은 직접 결과물을 만듭니다.
4. 하나의 실전 프로젝트 — AI 할일 SaaS "TaskFlow"를 기획부터 배포·회고까지 처음부터 끝까지 완성합니다.
5. 내부까지 — 공식 문서를 넘어 Codex 엔진의 실제 동작 원리(프로토콜·샌드박스·execpolicy·시스템 프롬프트)까지 파고듭니다.

## 관통 프로젝트: TaskFlow

이 책의 모든 실습은 한 방향을 가리킵니다 — 1인 개발자가 AI 에이전트만으로 굴리는 SaaS "TaskFlow"(AI 할일 관리 서비스). Part 4에서 기획·요구사항 정의(AGENTS.md + Goal)부터 백엔드(FastAPI)·프런트엔드·데이터베이스·인증·테스트 자동화·배포(Cloud/CI)·출시·회고까지 한 흐름으로 완성합니다. 앞선 Part에서 익힌 도구가 어디에 쓰이는지 실전에서 확인하게 됩니다.
